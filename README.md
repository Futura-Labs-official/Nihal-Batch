
## Run Locally

Clone the project ( use . to install project in current folder )

```bash
git clone https://github.com/Futura-Labs-official/restaurent-crud-react-sample .
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
